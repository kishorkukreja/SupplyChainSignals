# Inventory Timing Bet Carousel

Generated branded carousel assets for the Supply Chain Signals Medium article:

https://medium.com/@kishorkukreja/inventory-is-becoming-a-timing-bet-ea844592452a

## Files

- `slide-01.svg` through `slide-08.svg`: square 1080 x 1080 carousel slides.
- `preview.html`: browser preview of all slides.

## Brand

- Black / charcoal background.
- Off-white editorial typography.
- Vivid red signal accent.
- ECG-style signal line and briefing-footer motif.

## Export

Open `preview.html` to inspect the carousel. The SVG slides can be imported into Canva/Figma or converted to PNG/PDF for LinkedIn upload.
